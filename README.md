# base-react-js

## Pasos de instalación

1. Usar versión de Node 16.16.0
2. Clonar el repositorio.
3. Instalar el plugin de "editorconfig".
4. Instalar el plugin de "prettier".
5. Establecer prettier como el formateador predeterminado para archivos .js y .jsx.
6. En las configuraciones de VS Code, activar el formateo automático al guradar cambios de arhchivos.
7. Correr el comando `npm install`.
8. Correr el comando `npm start`.
